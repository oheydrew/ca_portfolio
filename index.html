projects

~/pixelhat/

pixelhat (or, as it’s more affectionately known, the ‘crappy doof hat’, ‘space cowboy’, or ‘can you please turn that thing off?’) is an open-source wearable Neopixel LED project, written in micropython on an ESP-8266 microprocessor.


This is a pet project, born out of a desire to learn micro- processors and programming, and out of my love of going out to festivals wearing silly hats.

~/boogle-books/

boogle-books is a terminal-based book library & information tool that uses a Google Books API. It’s a simple tool, which allows you to search through the API for titles, and displays the results in a tableized format, with extra single-title information available also.


The first piece of ruby code we ever wrote, this was an early project I co-coded with Mill Daulagala in the Coder Academy 2018 bootcamp.
